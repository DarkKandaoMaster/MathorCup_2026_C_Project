#### 第三步：Apriori关联规则挖掘

Apriori算法是关联规则挖掘的经典算法，用于从交易数据库中发现频繁项集和关联规则。

- **支持度（Support）**：项集X在所有交易中出现的频率
$$Support(X) = \frac{包含X的交易数}{总交易数}$$

- **置信度（Confidence）**：包含X的交易中同时包含Y的条件概率
$$Confidence(X \rightarrow Y) = \frac{Support(X \cup Y)}{Support(X)}$$

- **Lift值**：衡量X和Y的关联强度，Lift>1表示正相关，Lift=1表示独立，Lift<1表示负相关
$$Lift(X \rightarrow Y) = \frac{Confidence(X \rightarrow Y)}{Support(Y)}$$

Apriori算法工作原理：
1. 设定最小支持度阈值，先找出所有满足阈值的1-项频繁集
2. 由k-项频繁集生成(k+1)-项候选集
3. 扫描数据计算候选集支持度，剪枝保留频繁集
4. 重复直到无法生成更长的频繁集
5. 从频繁项集中提取满足最小置信度的关联规则

参数设置：
- 最小支持度：0.3（即特征组合至少在30%的高风险样本中出现）
- 最大项集长度：4
- 最小置信度：0.5

#### 第五步：核心特征组合识别与排名

在高风险组内通过Apriori算法发现的频繁项集中，筛选支持度≥0.35且长度≥2的高频组合，并计算**对比Lift**：
$$对比Lift = \frac{高风险组支持度}{非高风险组支持度}$$
对比Lift>1表示该组合在高风险组中比非高风险组更常见，值越大区分力越强。

#### 第六步：核心特征组合的医学解释

**【组合1】有饮酒 + 高TG(>1.92)（对比Lift = 13.65）**

- 有饮酒：酒精促进肝脏VLDL（极低密度脂蛋白）合成，加重高甘油三酯血症，且酒为湿热之品助湿生痰
- 高TG(>1.92)：甘油三酯升高是高血脂症最直接的血液标志物，痰湿体质者水湿代谢障碍导致膏脂内停，TG升高尤为显著

→ 综合解释：**脂质层面：甘油三酯升高直接反映膏脂内停；诱因层面：酒精促脂合成加重风险。** 饮酒与高TG的组合对比Lift高达13.65，意味着在痰湿体质高风险人群中，"有饮酒+高TG"这一组合的出现频率是非高风险组的13.65倍。机制上，酒精通过促进肝脏VLDL合成、抑制脂肪酸氧化，直接驱动TG升高；在痰湿体质内，酒精的"助湿生痰"效应与痰湿内蕴叠加，形成"饮酒→脂合成亢进→膏脂内停→高TG"的正反馈路径。

**【组合2】老年(≥60岁) + 高TC(>6.12)（对比Lift = 5.69）**

- 老年(≥60岁)：中老年脏腑功能衰退，脾胃运化力下降，气血运行减缓，痰湿更易内生，脂质代谢能力降低
- 高TC(>6.12)：总胆固醇升高反映整体脂代谢紊乱，痰湿内蕴致脾胃运化失常，膏脂积聚使TC升高

→ 综合解释：**脂质层面：总胆固醇升高直接反映膏脂内停；诱因层面：脏腑功能衰退加重风险。** 老年与高TC的组合对比Lift为5.69，在痰湿体质高风险人群中的出现频率是非高风险组的5.69倍。机制上，中老年人脾胃运化力下降，对脂质的代谢与排泄能力减弱，胆固醇在体内蓄积；痰湿体质本身的脾失健运与年龄相关的功能衰退叠加，使TC更容易超标。



#### 总结

1. 痰湿体质中高风险人群占71.9%（200/278人），说明痰湿体质是高血脂高风险的重要体质背景。
2. 卡方检验表明，高TG、高TC、高血尿酸是区分痰湿体质高/非高风险人群的三大显著特征（p<0.001），而人口学特征（年龄、性别、吸烟、饮酒）和活动量特征（ADL/IADL/活动量表总分）在痰湿体质内部不具有显著区分力。
3. Apriori关联规则挖掘结合对比Lift分析，识别出两个核心特征组合：
   - **有饮酒 + 高TG**（对比Lift = 13.65）：饮酒与高甘油三酯血症的协同效应在痰湿体质高风险人群中极为突出
   - **老年 + 高TC**（对比Lift = 5.69）：年龄相关的脂代谢衰退与高胆固醇血症在痰湿体质高风险人群中显著共现
4. 两个核心组合均呈现「**诱因（饮酒/衰老）→ 脂代谢紊乱（高TG/高TC）**」的链式病理机制，印证了痰湿体质高血脂高风险的病理本质是体质偏颇背景上的脂质代谢异常叠加。
