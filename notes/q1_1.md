### 问题1子任务1："从血常规体检指标、中老年人活动量表评分中，筛选出能有效表征痰湿体质严重程度、且能预警高血脂发病风险的关键指标"。

值得注意的是，从题目给的数据可以看出来，痰湿质是连续型变量，高血脂症二分类标签是分类型变量。
也就是说，我们需要根据输入变量 **血常规体检指标（TC、TG、LDL-C等）、中老年人活动量表评分（ADL各单项、IADL各单项）** ，找出同时和痰湿体质严重程度、高血脂发病风险最相关的几个指标。



#### 针对痰湿体质严重程度，这是一个回归问题，所以我们使用LASSO回归，通过L1正则化将不重要特征的系数压缩为0，留下来的即为关键指标。

LASSO 回归 = 传统的线性回归 + L1 正则化
- 损失函数（Loss Function）：
$$Loss = \sum_{i=1}^n (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^p |\beta_j|$$
前半部分是残差平方和（要求模型预测得越准越好）；后半部分就是 L1 惩罚项（要求留下的指标越少越好）。$\lambda$ （Lambda）是你可以调节的超参数。
- 工作原理：
1. 当 $\lambda = 0$ 时，它就是普通的线性回归，所有输入指标都会有一个权重。
2. 随着你慢慢增大 $\lambda$，惩罚力度变强。
3. 那些对"痰湿积分"影响较小、或者与其他指标高度重合的冗余指标，其回归系数 $\beta$ 会被迅速压缩到 0。
4. 最终模型中系数不为 0 的指标，就是经过 LASSO 严选出来的、对痰湿积分最具预测价值的"关键指标"。

为什么选择LASSO回归？
1. 数据没有任何缺失值、异常值。（LASSO回归最大的缺点是怕异常值）
2. LASSO考虑了变量之间的相互作用，得出的结果在数学上更具说服力。
3. 解决多重共线性：医学或生物指标（如体检数据）往往高度相关（例如体重、腰围、BMI同时存在）。如果只用相关系数筛选，这些高度重合的指标会被一起选入模型，导致后续的回归模型崩溃（共线性问题）。LASSO 能够凭借全局视角，在这些冗余指标中只保留最关键的一个，将其他指标的系数精准压缩为 0。

三种alpha准则对比：
| 准则            | alpha  | 选中特征数 |
|-----------------|--------|------------|
| 最小MSE         | 0.0051 | 2          |
| 1-SE            | 0.0010 | 13         |
| 宽松(alpha*0.1) | 0.0005 | 17         |

宽松准则下选出13个血常规+活动量表指标，Top5为：
ADL吃饭（负向, -0.068）— 活动量表
TC总胆固醇（负向, -0.048）— 血常规
IADL交通（正向, 0.024）— 活动量表
LDL-C低密度脂蛋白（正向, 0.020）— 血常规
ADL洗澡（正向, 0.016）— 活动量表
需要注意的是：R² = 0.0146，说明这些指标对痰湿质的线性解释力较弱，写论文时需讨论。



#### 针对高血脂发病风险，这是一个分类问题，所以我们构建随机森林（Random Forest）分类模型，利用模型输出的"特征重要性评分（Feature Importance）"来客观排序，挑选排名靠前的指标。

随机森林 = 多棵决策树集成 + Bagging（自助法采样）
- 核心思想：对训练数据集进行有放回抽样（Bootstrap），每棵树在随机选取的特征子集上独立生长，最终通过投票（分类）或平均（回归）得到预测结果。
- 特征重要性（Mean Decrease in Impurity, MDI）：
$$Importance(j) = \frac{1}{N_{trees}} \sum_{t=1}^{N_{trees}} \sum_{n \in Nodes(j)} \Delta Gini(n)$$
即特征 $j$ 在所有树的所有节点中，带来的不纯度下降（基尼增益）之和，再对所有树取平均。MDI 值越大，说明该特征对分类决策的贡献越关键。
- 工作原理：
1. 每棵树在训练时只使用一部分样本和一部分特征，这保证了模型不容易过拟合。
2. 每次分裂时，算法遍历当前可用的特征子集，选择使基尼不纯度下降最大的特征作为分裂点。
3. 特征被选为分裂点的次数越多、带来的基尼增益越大，其重要性评分就越高。
4. 最终将所有树的特征重要性取平均，即可得到每个特征的总体重要性排名。

为什么选择随机森林？
1. 能自动处理非线性关系：血脂指标之间（如TG与TC）往往存在复杂的交互效应，决策树天然支持非线性分裂，无需手动构造交互项。
2. 对共线性不敏感：随机森林每次分裂只选一个特征，冗余特征不会同时获得高分，天然具备特征筛选能力。
3. 不需要标准化：树模型基于排序分裂，对特征的量纲不敏感，与前面LASSO的归一化要求互为补充。
4. 适合高维筛选场景：随机森林给出的MDI重要性可以快速定位最关键的少数指标，与LASSO的"稀疏化"思路形成互补验证。

结果：AUC = 1.0，前4个特征即达80%累积重要性：
| 指标            | 重要性 | 累积   |
|-----------------|--------|--------|
| TG甘油三酯      | 0.34   | 0.34   |
| TC总胆固醇      | 0.29   | 0.63   |
| 血尿酸          | 0.16   | 0.79   |
| LDL-C           | 0.07   | 0.86   |
| HDL-C           | 0.05   | 0.91   |

AUC = 1.0 是预期结果——血脂四项（HDL/LDL/TG/TC）本身就是高血脂的诊断依据，模型能完美区分属于正常现象，恰恰验证了数据的一致性。



#### 综合分析

LASSO 回归针对"痰湿体质严重程度"（回归任务），随机森林针对"高血脂发病风险"（分类任务），两者从不同角度筛选指标。我们取两种方法共同选中的指标，即为"同时表征痰湿质且预警高血脂"的关键指标。

综合评价方式：将 LASSO 系数绝对值和 RF 特征重要性分别归一化到 [0,1]，求和得到综合得分，按得分降序排列。

两种方法同时选中的5个关键指标：
| 指标             | 类型   | LASSO方向 | RF重要性排名 |
|------------------|--------|-----------|--------------|
| TC（总胆固醇）   | 血常规 | 负向      | 第2          |
| LDL-C（低密度脂蛋白） | 血常规 | 正向 | 第4          |
| TG（甘油三酯）   | 血常规 | 正向      | 第1          |
| 空腹血糖         | 血常规 | 正向      | 第6          |
| HDL-C（高密度脂蛋白） | 血常规 | 正向 | 第5          |

值得注意的是，5个关键指标全部来自血常规，活动量表评分虽在 LASSO 中有若干入选，但在 RF 重要性排名中靠后，说明活动量表指标与高血脂发病风险的直接关联弱于血常规指标。
